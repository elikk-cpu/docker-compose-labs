# Подводные камни

- По умолчанию Flask-приложение слушает `127.0.0.1`.
- Если сначала выполнить `COPY . .`, изменение любого файла инвалидирует cache.
- Shell-форма или неверный executable в CMD могут привести к неправильной обработке
  сигналов или немедленному завершению.
- Файл `.env` не должен попадать в build context.
