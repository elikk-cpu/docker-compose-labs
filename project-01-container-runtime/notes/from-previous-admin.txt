Требования:
- сайт должен открываться с хоста на порту 8088;
- внутри Docker-сети сайт должен быть доступен по имени landing;
- сайт и конфиг нельзя изменять из контейнера.

Известные ошибки предыдущего запуска:
- контейнер сразу завершался;
- наружу публиковали не тот внутренний порт;
- соседний контейнер пытался обращаться к landing через localhost.
